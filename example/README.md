# example
This project was created using the `vue create` command for E2E testing.

## Project setup
```
yarn install
```

### Run Server

```sh
# dev
yarn run dev

# prod
yarn run start

# Koa
yarn run koa

# Express
yarn run express
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test:e2e
```

### Lints and fixes files
```
yarn run lint
```
